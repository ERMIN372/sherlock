# Sherlock — AI Face Search

Веб-версия сервиса поиска похожих лиц. Пользователь загружает фотографию,
обрезает лицо и получает визуально похожие изображения из публичных веб-источников
через **официальный сторонний Face Search API** (FaceCheck.ID).

> Сервис не скрейпит социальные сети и не обходит ограничения платформ. Все
> результаты приходят из стороннего API. Загруженные изображения не сохраняются
> на сервере.

## Возможности

- Загрузка, предпросмотр и **обрезка лица** (drag&drop, зум, круглая область).
- Отправка изображения в Face Search API через серверный прокси.
- Отображение результатов: миниатюры, **процент сходства**, ссылки на источники.
- Состояния: ожидание, поиск, успех, **нет результатов**, ошибка.
- **История запросов** (хранится локально в браузере).
- **Система кредитов** и **демонстрационная оплата** (без реальных платежей).
- Удаление загруженного изображения после завершения поиска (нет записи на диск).
- **Ограничение частоты запросов** (rate limit по IP) и предупреждение о
  допустимом использовании.
- Адаптивный современный интерфейс (Next.js + Tailwind).

## Архитектура

- **Next.js 15 (App Router) + TypeScript + Tailwind v4** — единое приложение.
- Абстракция **`FaceSearchProvider`** (`lib/providers/types.ts`) с двумя реализациями:
  - `FaceCheckProvider` (`lib/providers/facecheck.ts`) — реальный REST-поток
    FaceCheck.ID (`upload_pic` → polling `search`).
  - `DemoProvider` (`lib/providers/demo.ts`) — детерминированный мок без сети,
    включается автоматически, если не задан API-ключ.
- `app/api/search/route.ts` — серверный прокси: валидация, rate limit, вызов
  провайдера, изображение держится только в памяти на время запроса.
- `app/api/payment/route.ts` — демо-оплата (без реальных денег).

## Быстрый старт

```bash
npm install
cp .env.example .env.local   # по желанию; без ключа работает демо-режим
npm run dev                  # http://localhost:3000
```

Production:

```bash
npm run build && npm start
```

## Переменные окружения

| Переменная | Назначение | По умолчанию |
| --- | --- | --- |
| `FACE_SEARCH_PROVIDER` | `facecheck` / `demo`. Пусто = авто-выбор | авто |
| `FACECHECK_API_TOKEN` | Токен FaceCheck.ID. Без него — демо-режим | — |
| `FACECHECK_API_URL` | Базовый URL API | `https://facecheck.id` |
| `FACECHECK_TESTING_MODE` | `true` — тест-режим FaceCheck (без кредитов) | `false` |
| `RATE_LIMIT_MAX` | Макс. запросов в окне на один IP | `10` |
| `RATE_LIMIT_WINDOW_MS` | Длина окна rate limit (мс) | `60000` |

## Допустимое использование

Инструмент предназначен только для законного и авторизованного использования
(например, поиск собственного лица или с согласия человека). Запрещены
преследование, слежка, деанонимизация и любое нарушение приватности и условий
платформ.
